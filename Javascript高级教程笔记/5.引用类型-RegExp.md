### 创建方式
```
var reg = new RegExp('[0-9]','ig');
var reg = /[0-9]/ig;			//同上，推荐
```

### 测试方法
```
var reg = /[0-9]/;
//a. test()
var test = reg.test('a22');
console.log(test);			//true

//b. exec()
var exec = reg.exec('a22sd124');
console.log(exec);		//返回一个数组[0:'2', index:'1', input:'a22sd124']
```

### 修饰符
```
a. g  全局匹配
b. i  忽略大小写
c. m  允许多行匹配,主要针对^$在换行时
```

### 贪婪于惰性
```
贪婪限定符：+ ? * {n,} {n,m}        //尽可能匹配多的字符
惰性限定符：+? ?? *? {n,}? {n,m}?   //加一个?,会尽可能少匹配 
```
### 常用正则

#### 1. 匹配数字和字符串
```
整数和小数： /^-?(0|[1-9]\d*)(\.\d+)?$/  //0 和 [1-9]/d* 为两部分
只能输入数字： /^\d*$/
只能输入n位数字： /^\d{n}$/
只能输入至少n位数字： /^\d{n,}$/
只能输入n-m位数字： /^\d{n,m}$/
只能输入0和非0开头的数字： /^(0|[1-9]\d*)$/
只能输入有两位小数的正实数： /^(0|[1-9][0-9]*)(\.\d{2})?$/
只能输入有1-3位小数的正实数： /^(0|[0-9][1-9]*)(\.\d{1,3})?$/
只能输入非0的正整数： /^[1-9][0-9]*$/
只能输入非0的负整数： /^-[1-9][0-9]*$/
只能输入长度为3的字符： /^.{3}$/
只能输入英文字母： /^[a-zA-Z]+$/
只能输入大写字母： /^[A-Z]+$/
只能输入小写字母： /^[a-z]+$/
只能输入字母和数字： /^[a-zA-Z0-9]+$/
只能输入数字字母和下划线： /^\w+$ / (绿叶学习网讲解有误，\w是不能匹配中文的)
```
#### 2. 验证用户信息
```
验证用户密码，字母开头，长度在8-16之间，只能包含字母，数字和下划线： /^[a-zA-Z]\w{7,15}$/
验证是否含有 ^%&',;=?$\"等字符： /^[\^%&',;=?*\\]+$/
检验中文： /^[\u4e00-\u9fa5]+$/   //unicode编码
验证Email地址： /^\w+@[a-zA-Z]+\.(com|cn|com\.cn){1}$/
验证InternetURL：/^https?:\/\/[\w\.&-\/?=#]+$/
验证手机号码： /^1[358]\w{9}$/
匹配首位空格： /(^\s*)|(\s*$)/g/
